# 2026 奇萊東稜水源計畫

> 本文件依奇萊東稜常見水源、太管處路標紀錄與山友行程紀錄整理。水源會受梅雨、颱風前後、連續晴天與隊伍數量影響；出發前 3 天須查近期回報，並以最終 GPX 標定水源點。

---

## 1. 用水原則

| 原則 | 說明 |
|---|---|
| 池水必濾煮 | 月形池、驚嘆號水池、黑水塘、平安池等皆須過濾，必要時煮沸 |
| 不把乾水池列為水源 | 2818乾水池只能當地名與路線點，不能作為補水依據 |
| 先補水再休息 | 抵達營地或水源點後，先確認水況與隔日水量 |
| 缺水時取消山頭 | 太魯閣大山、帕托魯山皆可因缺水取消 |
| 每人保留底線水 | 任何時間每人至少保留 0.8～1.0 L 安全水 |

| 研海林道水源 | 中～高 | 部分溪溝為活水 | 中 | D4 行動水，每人 1.5～2.0 L | 若林道水源落空，不停留煮食 | 雨後可能混濁，注意落石與蜂類 |

## 3. 必須背水路段

| 路段 | 建議起登水量 / 人 | 背水原因 | 決策 |
|---|---:|---|---|
| 黑水塘 / 成功山屋 → 驚嘆號水池 | 2.5～3.0 L | 高海拔爬升、曝曬與稜線風 | 成功山屋若無水，D1 節水 |
| 驚嘆號水池 / 磐石西峰黑水塘 → 磐石中峰 | 3.0 L | 磐石中峰水源需下切，耗時且不穩 | 抵達前先背足晚餐與早餐水 |
| 磐石中峰 → 三叉營地 | 3.0 L | 白石附近水源不可保證 | D2 起登前就當中途無水規劃 |
| 三叉營地 → 平安池 → 立霧主山後段 | 3.0～3.5 L | D3 長距離且可用水點不穩 | 平安池補不足即取消帕托魯山 |
| 三岔路營地 → 研海林道12K | 2.0～2.5 L | 長下降、疲勞與摸黑風險 | 每人保留 0.8～1.0 L 到工寮 |

## 5. 每人最大攜水量

| 條件 | 最大攜水量 / 人 | 適用情境 |
|---|---:|---|
| 一般行進 | 2.0～2.5 L | 水源相對可期待、天氣涼爽 |
| 長距離水源不穩 | 3.0 L | D1、D2 多數路段 |
| 枯水或高溫 | 3.5 L | D3 或近期回報缺水 |
| 極端缺水備案 | 4.0 L | 僅必要時使用，需評估負重造成的速度下降 |

## 7. 濾水與消毒

| 裝備 | 建議數量 | 備註 |
|---|---:|---|
| 個人濾水器 | 每人自備或兩人共用 1 組 | 需事前自行確認可正常出水 |
| 備用濾水器 | 至少 1 組 | 建議由領隊或副領隊確認 |
| 淨水錠 | 每人至少 6～10 錠 | 池水、濾水器故障、緊急備案 |
| 折疊水袋 | 依帳篷單位準備 | 晚餐與早餐水 |

## 8. 缺水備案

| 情境 | 判斷條件 | 處置 |
|---|---|---|
| 驚嘆號水池水差 | 水少、混濁、取水困難 | 補最低安全量，D1 後段節水，磐石西峰黑水塘再判斷 |
| 磐石中峰水源取水困難 | 下切耗時過長或水量不足 | D2 取消不必要停留，起登每人至少 3.0 L |
| 三叉營地水量不足 | 晚餐與 D3 起登水不足 | 取消太魯閣大山或縮短往返 |
| 平安池補不足 | 每人無法補至 3.0 L 以上 | D3 取消帕托魯山，保守推進研海林道 |
| 2818乾水池無水 | 預期狀況 | 不停留煮食，確認每人安全水 |
| 研海林道水源落空 | 到 11K 附近仍無穩定水 | D4 不煮食，集中下山 |
