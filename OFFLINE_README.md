# 奇萊東稜離線閱讀方式

## 最穩方式：單檔離線版

請把 `Kilai-East-Ridge-2026-offline.html` 傳到手機，並儲存在手機的「檔案 App」。

| 步驟 | iPhone 操作 |
|---|---|
| 1 | 用 AirDrop 接收 `Kilai-East-Ridge-2026-offline.html` |
| 2 | 選擇「儲存到檔案」 |
| 3 | 出發前開啟一次，確認圖片、行程、水源、緊急資訊都正常 |
| 4 | 山上沒網路時，從「檔案 App」直接點開 |

## 公開網站離線快取

若使用 GitHub Pages 或 Netlify 網址，請隊員在有網路時先打開一次網站。網站已加入離線快取設定，開過一次後，手機瀏覽器通常可在無網路時再次開啟同一頁。

| 注意 | 說明 |
|---|---|
| 必須先開過 | 沒網路前若從未打開過，手機不會有快取 |
| 不如單檔穩 | 瀏覽器可能因空間或系統策略清快取 |
| 建議做法 | 公開連結 + 單檔離線版兩個都準備 |
