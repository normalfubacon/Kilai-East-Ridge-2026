# 2026 奇萊東稜緊急應變計畫

> 核心原則：先確保現場安全，再處理傷患；先穩定生命徵象，再決定撤退或求援；所有重大異常須回報留守人查爾斯。

## 2. 失溫處理流程

| 階段 | 徵象 | 處置 |
|---|---|---|
| 輕度失溫 | 發抖、手腳冰冷、反應變慢 | 停止行進、加衣、補熱量、避風避雨 |
| 中度失溫 | 劇烈發抖或停止發抖、語無倫次、步態不穩 | 換乾衣、睡墊隔地、保溫毯包覆、熱飲少量多次 |
| 重度失溫 | 意識不清、脈搏弱、無法自行行走 | 立即通報救援，避免粗暴搬動，持續保溫 |

---

## 3. 高山症處理流程

| 等級 | 判斷 | 處置 |
|---|---|---|
| 輕度 AMS | 頭痛、噁心、食慾差、睡不好 | 停止加速、補水、保暖、觀察 |
| 中度 AMS | 頭痛加劇、嘔吐、虛弱、步態不穩 | 停止上升，取消山頭，評估下降 |
| HACE 疑似 | 意識混亂、走路像喝醉、嗜睡 | 立即下降並通報救援 |
| HAPE 疑似 | 喘、咳嗽、胸悶、休息仍呼吸困難 | 立即下降、保暖、通報救援 |

## 4. 蜂螫與過敏性休克

| 情境 | 徵象 | 處置 |
|---|---|---|
| 單一蜂螫、局部紅腫 | 疼痛、紅腫、搔癢 | 遠離現場，冰敷，口服抗組織胺，觀察 |
| 多處蜂螫 | 多處疼痛、腫脹、頭暈 | 立即離開蜂群區，通報留守，評估撤退 |
| 過敏性休克疑似 | 呼吸困難、喉嚨緊、全身蕁麻疹、嘴唇舌頭腫、低血壓、意識改變 | 使用個人 EpiPen，立即通報 119 / SOS，保暖並準備撤離 |

| EpiPen 原則 | 說明 |
|---|---|
| 是否需要攜帶 | 曾有蜂螫過敏、嚴重食物 / 藥物過敏或醫師建議者，應依處方自備 |
| 未知過敏史者 | 建議備抗組織胺與類固醇藥膏；EpiPen 為處方藥，不建議未經醫囑任意準備 |
| 隊伍管理 | 有過敏史者須出發前告知 Pony 與小宇，並說明 EpiPen 放置位置 |

## 5. 螞蝗與硬蜱處理

| 風險 | 常見位置 | 預防 |
|---|---|---|
| 螞蝗 | 研海林道、中低海拔潮濕草叢、雨後路段 | 綁腿、長褲塞襪、休息時檢查襪口 |
| 硬蜱 | 箭竹、草叢、獸徑、林道邊坡 | 淺色長袖長褲、避免坐草地、每日睡前全身檢查 |

| 狀況 | 處置 |
|---|---|
| 螞蝗叮咬 | 移除後清潔傷口，加壓止血，保持乾燥，勿抓傷 |
| 硬蜱附著 | 用尖頭鑷子貼近皮膚夾住，穩定垂直拉出；不要用火燒或塗油 |
| 傷口紅腫發熱 | 標記時間與範圍，下山後就醫 |

## 6. 墜落 / 外傷處理

| 類型 | 可能狀況 | 處置 |
|---|---|---|
| 擦挫傷 | 表皮破損、流血 | 清潔、止血、包紮，評估是否續行 |
| 扭傷 | 腳踝、膝蓋疼痛腫脹 | RICE 原則，彈繃固定，減重，評估撤退 |
| 骨折疑似 | 變形、劇痛、無法承重 | 固定患部，不強迫行走，通報救援 |
| 頭部撞擊 | 頭暈、嘔吐、嗜睡 | 視為嚴重，停止行進並通報 |
| 墜落 | 多重外傷、脊椎風險 | 不任意搬動，確認呼吸與意識，通報救援 |

## 7. 撤退原則

| 原則 | 說明 |
|---|---|
| 人命優先 | 山頭可取消，行程可變更 |
| 不分散隊伍 | 除非通訊與安全條件充分，否則不拆隊 |
| 不摸黑硬闖危險地形 | 鐵線斷崖、崩壁、裸岩、研海林道崩塌段應避免夜間通過 |
| 傷患不可落單 | 至少 2 人陪同，並保持通訊 |
| 每次決策均回報 | 撤退方向、座標、預計抵達時間、傷患狀態 |

| 撤退觸發條件 | 行動 |
|---|---|
| 重度高山症疑似 | 立即下降，通報救援 |
| 外傷無法自行穩定行走 | 原地保暖固定，通報救援 |
| 過敏性休克疑似 | EpiPen 後立即通報救援 |
| 水源落空造成安全風險 | 取消山頭，改走最短安全路線 |
| 天候惡化無法通過暴露路段 | 等待、退回安全營地或通報 |

## 9. 119 報案資訊格式

|---|---|
| 報案人 | 我是 2026 奇萊東稜隊伍 Pony |

| 海拔 | 約 3,000 公尺 |
| 人數 | 全隊 6 人，1 人受傷 |
| 傷勢 | 右腳疑似骨折，無法自行行走，意識清楚 |
