## 4. 危險路段

| 路段 | 風險 | 通過原則 |
|---|---|---|
| 奇萊北峰稜線 | 強風、低溫、高山症 | 身體不適立刻說 |
| 鐵線斷崖 | 暴露、落石、濕滑 | 全隊集中，逐一通過，不停留拍照 |

| 大理石裸岩 | 雨天濕滑 | 登山杖輔助，降低速度 |
| 研海林道 | 崩塌、落石、草深、螞蝗、蜂類 | 不單獨行動，不摸黑硬闖 |

## 6. 裝備重點

| 必備 | 說明 |
|---|---|
| 雨衣雨褲 | 需可長時間行進 |
| 睡袋、睡墊 | 高山夜間保暖，D3 需自備營帳睡眠系統 |
| 頭燈與備電 | D3 很可能長時間行進 |
| 離線地圖與 GPX | 手機飛航模式下仍須可使用 |
| 行動電源 | 建議 10,000～20,000 mAh |
| 保暖層與羽絨 | 稜線、營地、雨天失溫風險高 |
| 綁腿、長褲、手套 | 箭竹、螞蝗、硬蜱、岩稜皆需要 |
| 濾水器 / 淨水錠 | 個人須能處理池水 |
| 抗組織胺 | 蜂螫與過敏備用 |
| EpiPen | 有蜂螫或嚴重過敏史者依醫囑自備 |
| 個人保險 | 自行申請並留存資料 |

## 7. 撤退原則

| 情境 | 原則 |
|---|---|
| 高山症惡化 | 不再上升，必要時立即下降 |
| 外傷無法穩定行走 | 停止行程，保暖固定，通報 |
| 雷雨或強風 | 暴露路段不硬闖 |
| 水源落空 | 取消太魯閣大山或帕托魯山，重新分配水量 |
| 行程嚴重延誤 | 優先安全住宿，不摸黑通過危險地形 |
| 蜂群攻擊或過敏反應 | 先離開現場，疑似休克即通報救援 |
